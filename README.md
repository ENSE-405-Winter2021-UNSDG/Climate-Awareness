# ENSE-405-Community_car_pool
mobile application that lets highschool and post-secondary studens share ride to school. 
this application is intened with the hope of reducing emssion to support goal 13 of UN's [sustainable Development](https://www.un.org/sustainabledevelopment/climate-change/) goal.
